# Demonstration Artifacts: RISE RO-Crate Profile

This repository contains the demonstration artifacts for the paper **"RISE: An RO-Crate Profile for Standardizing Research Infrastructure Experimental Artifacts"**.
The paper was authored by Eric Hauser, Kilian Warmuth, Marcel Kempf, Lennard Harte, and Georg Carle. It has been accepted at the 2026 ACM Conference on Reproducibility and Replicability (ACM REP'26).

The visualized artifacts of the two demonstration experiments are available on GitHub Pages:
- [Demonstration Experiment 1 - QUIC Measurement](https://tumi8.github.io/paper-rise-ro-crate-profile-demonstration/experiment1)
- [Demonstration Experiment 2 - PQC Reproduction](https://tumi8.github.io/paper-rise-ro-crate-profile-demonstration/experiment2)
